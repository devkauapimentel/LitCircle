# Projetos Futuros — Evolução LitCircle e Além

> Cada projeto constrói em cima do anterior. A ordem importa.

---

## Nível 2 — Evolução do LitCircle (após MVP v0)

### LitCircle v1.0 — Features completas
- Biblioteca de livros pessoal (CRUD)
- Timer Pomodoro com streak (calendário estilo GitHub)
- Sistema de rodízio de escolha de livro
- Calendário de sessões
- Perfil do usuário com foto

### LitCircle v2.0 — Migração para React
- Frontend reescrito em React + Vite
- TailwindCSS para estilização
- React Query para cache de API
- React Router para navegação SPA
- **O que aprende:** Componentização, estado, hooks, build tools

### LitCircle v3.0 — Infraestrutura profissional
- Docker + docker-compose para ambiente local
- Redis para cache de chat e sessões ativas
- GitHub Actions CI/CD com testes automáticos
- Testes: Jest (Node), JUnit (Java), Vitest (React)
- **O que aprende:** Containers, cache, CI/CD, testes automatizados
